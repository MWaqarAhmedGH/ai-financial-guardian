# 🛡️ AI Maliati Dost (AI Financial Guardian)

**Pakistan's Premium AI Financial Safety Assistant**  
*Built for the Google Antigravity Hackathon 2026*

---

## 🚩 The Problem
Pakistan faces a massive surge in financial fraud via SMS, WhatsApp, and fake calls. Many citizens lose their hard-earned money because they cannot distinguish between official bank messages and sophisticated scams (BISP, Lottery, SIM block). Additionally, there is a lack of simple, Urdu-based financial literacy and budgeting tools tailored for the Pakistani lifestyle.

## 🚀 The Solution
**AI Maliati Dost** is an all-in-one financial guardian that uses Google's Gemini AI to protect users. It provides:
- **Multimodal Scam Detection:** Scan screenshots of suspicious messages.
- **Urdu Voice Support:** Accessibility for users with low digital literacy.
- **Smart Budgeting:** Interactive tools to manage local income (50/30/20 rule).
- **Direct Reporting:** One-click connection to FIA Cybercrime authorities.

---

## ✨ Key Features
- 📸 **AI Screenshot Scan:** Extract text and detect fraud in JazzCash/Bank screenshots.
- 🎤 **Urdu Voice Interaction:** Talk to the AI in Urdu; it replies with a human-like voice.
- ⚠️ **Real-time Scam Ticker:** Live news alerts on trending Pakistani scams.
- 📊 **Interactive Budget Planner:** Visual breakdown of monthly expenses and savings.
- 🛑 **Scam Risk Scoring:** Immediate percentage-based danger assessment.
- 🚨 **One-Click Reporting:** Direct link to FIA Cybercrime complaint portal.
- 📱 **PWA Support:** Installable on any mobile or desktop as a native app.
- 💾 **Chat Persistence:** Never lose your advice; chat history is saved locally.

## ⚙️ Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/MWaqarAhmedGH/ai-financial-guardian.git
   cd ai-financial-guardian
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Configure Environment Variables:**
   Create a `.env` file in the root directory:
   ```env
   GEMINI_API_KEY_1=your_api_key_here
   GEMINI_API_KEY_2=your_api_key_here
   PORT=3000
   ```

4. **Run the application:**
   ```bash
   npm start
   ```
